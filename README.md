# Design Pattern in Swift

Most common Cocoa design patterns: 

- Creational: Singleton
- Structural: MVC, Decorator, Adapter, Facade
- Behavioral: Observer, and, Memento 

 has been implemented in this Music Album App

### Version
- XCode 7.3
- iOS 9.3
- Swift 2.2
